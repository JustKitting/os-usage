# Level Ideas

## Fundamental Form Elements
- [x] Slider / range input — drag to a specific value (volume, price filters, settings) → Level 16
- [x] Radio buttons — pick one from a group → Level 17
- [x] Number stepper — +/- buttons to increment/decrement (e-commerce qty) → Level 18
- [x] Star rating — click the Nth star (reviews, feedback) → Level 19

## Navigation Patterns
- [x] Tabs — click tabs to switch panels, find/interact with content in a specific tab → Level 20
- [x] Accordion / collapsible sections — expand the right section, interact with hidden content → Level 21
- [x] Modal / dialog — popup that needs confirm, dismiss, or fill out → Level 22
- [x] Context menu — right-click to open menu, pick the right option → Level 23

## Compound Interactions
- [x] Search / autocomplete — type query, select from suggestion dropdown → Level 24
- [x] Sortable list — drag items to reorder them → Level 25
- [x] Multi-select tags — click items that appear as removable chips → Level 26
- [x] Toast / notification dismiss — find and close the right notification → Level 27
